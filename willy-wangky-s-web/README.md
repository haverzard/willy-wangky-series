# Willy Wangky's Web
## Description
Website application for selling chocolate built using PHP without framework.

## Requirements
- MySQL
- PHP >= 7.0
- WS-Supplier
- WS-Factory

## How to Use
Run `php -S localhost:<port> -t <path/to/folder/public>`
